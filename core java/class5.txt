Steps to setup environmental variables 
======================================
step1:
-----
	Make sure JDK 24 installed successfully.

step2:
------
	Copy "bin" directory from "java_home" folder.
	ex:
		C:\Program Files\Java\jdk-24\bin

step3:
------
	Paste "bin" directory in environmental variables.
	ex:
		Right click to This PC --> properties --> Advanced system settings 
		---> environmental variables ---> system variables --> click new button
		---> variable name : path 
		     variable value : C:\Program Files\Java\jdk-24\bin;
		--> ok --> ok --> ok.

step4:
------
	Check environmental setup done perfectly or not.
	ex:
		cmd> javap 
		cmd> java -version 


Steps to develop first java application 
=======================================
step1:
------
	Create a "javaprog" folder inside 'D' drive.

step2:
------
	Open the notepad and write Hello World program.
	ex:
		class Test
		{
			public static void main(String[] args)
			{
				System.out.println("Hello World");
			}
		}
step3:
------
	Save java program with same name as class inside "javaprog" folder.

step4:
------
	Open the command prompt from "javaprog" location.

step5:
------
	Compile the java program by using below command.
	ex:
		javac    Test.java
				|
				filename 

step6:
------
	Execute the java program by using below command.
	ex:
		java     Test 
			   |
			 classname
