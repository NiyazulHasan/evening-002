Types of objects in java
=========================
We have two types of objects in java.

1) Immutable object 

2) Mutable object 

1) Immutable object 
---------------------
After object creation if we perform any changes then for every change a new object will be created such type of object is called immutable object.
ex:
	String and Wrapper classes

2) Mutable object
----------------
After object creation if we perform any changes then all the required changes will be done in a same object such type of object is called mutable object.
ex:
	StringBuffer and StringBuilder


String 
======
It is a collection of characters which is a enclosed in a double quotation.

case1:
------
Once if we create a String object we can't perform any changes. If we perform any changes 
then for every change a new object will be created such behaviour is called immutability of an object.

ex:
---
class Test  
{
	public static void main(String[] args) 
	{
		String s = new String("ihub");
		System.out.println(s); // ihub 
		
		s.concat("talent");
		System.out.println(s); // ihub
	}
}

Difference between == and .equals() method?

==
---
It is a equality operator that returns boolean value either true or false.
It is used for reference comparision or address comparision.
ex:
class Test  
{
	public static void main(String[] args) 
	{
		String s1 = new String("ihub");
		String s2 = new String("ihub");
		System.out.println(s1 == s2); // false 
	}
}


.equals()
---------
It is a predefined method that returns boolean value either true or false.
It is used for content comparision which is case sensitive.

ex:
---
class Test  
{
	public static void main(String[] args) 
	{
		String s1 = new String("ihub");
		String s2 = new String("ihub");
		System.out.println(s1.equals(s2)); // true
	}
}

Q) Write a java program to find out number of digits and perform sum of digits then display it is even or odd?

input:
	NEX123

output:
	No of Digits : 3
	Sum of Digits : 6
	It is even number 

class Test  
{
	public static void main(String[] args) 
	{
		String str="NEX123";
		
		str = str.replaceAll("[^0-9]","");
		
		System.out.println("No of Digits :"+str.length());
		
		int sum=0;
		for(int i=0;i<str.length();i++)
		{
			int n = Character.getNumericValue(str.charAt(i));
			sum += n;
		}
		System.out.println("Sum of Digits :"+sum);
		if(sum%2==0)
			System.out.println("It is even number");
		else
			System.out.println("It is odd number");
	}
}


Q) Write a java program to insert given word in a given string?

input:
	str = JavaStudent
	index = 4
	word = for 

output:
	JavaforStudent


class Test  
{
	public static void main(String[] args) 
	{
		String str = "JavaStudent";
		int index = 4;
		String word = "for"; 

		String word1 = str.substring(0,index); //java
		
		String word2 = str.substring(index); // student
		
		str = word1+word+word2;
		System.out.println(str);
	}
}
